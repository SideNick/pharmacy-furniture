# Product

## Name
Pharmacy Furniture QR

## Goal
A web service for tracking pharmacy furniture, confirming presence via QR, reporting damage, and automatically creating tasks in Bitrix24.

## Primary users
- designer / admin
- pharmacy employee
- service or operations team

## Main problem
Furniture is installed in pharmacies, but inventory checks and damage reporting are manual, slow, and inconsistent.

## Core value
Each furniture item has a unique QR code. A pharmacy employee scans it, confirms presence or reports a problem, and the system records the event and can create a Bitrix24 task automatically.

## MVP scope
- create pharmacies
- create furniture types
- create furniture items
- generate QR for each item
- public item page after scanning
- confirm presence
- report damage with comment and photo
- create Bitrix24 task through webhook

## Not MVP
- predictive analytics
- warehouse ERP sync
- advanced BI dashboards
- multi-system integrations
